# 🚀 AI Sales Forecasting System

## Overview
End-to-end ML system for retail sales forecasting with drift detection and retraining.

## Features
- Time-series feature engineering
- Multi-model training (XGBoost, RF, etc.)
- MLflow tracking
- Drift detection
- Retraining pipeline
- Streamlit dashboard

## Tech Stack
Python, MLflow, Streamlit, XGBoost, Evidently AI

## How to Run
pip install -r requirements.txt  
python train.py  
streamlit run main.py
